Item Price Update

The purpose of this scheduled job is to update the Catalog Item price. The scheduled Job will executes the stored procedure every day at 12:00:00 AM.

Scheduled Job Setup


Dependencies

1. The Catalog Item Id, Price and Effective Date must be defined in a look up table (PurCatalogItemPriceUpdateSchedules)
2. Scheduled Job must be enabled for updating the catalog item price.

Database Objects

Object Types Object Names
Tables PurItems
PurCatalogItems
PurCatalogItemPriceUpdateSchedules
Stored Procedures PurCatalogItemPriceUpdate
