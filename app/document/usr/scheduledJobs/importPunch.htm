Kronos Punch Import

The purpose of this scheduled job is to import the punches from old TeamFin database. The scheduled Job will execute the sql query every 10 minutes.

Scheduled Job Setup


Dependencies

1. Employees should be available in a look up table (Employees - Refer old TeamFin database)
2. Punches should be available in a look up table (EmpPunches - Refer old TeamFin database)
3. Employee Punch Types must be defined in a system look up table (PayEmployeePunchTypes)
4. Device Group Types must be defined in a system look up table (EmpDeviceGroupTypes)
5. Employees should be available in a look up table (EmpEmployeeGenerals)
6. Scheduled Job must be enabled for importing the punches fron old TeamFin database.
