Action

1. This user interface can be accessed at Configuration > People.
2. Edit Person - Enter Last, First, or Brief of Person and click Search button. It allows User to edit Person.
3. Save Person -  Click on Save to save the change.
4. Undo current changes to Person - Click Undo to cancel any changes made to selected Person.
