Skill

This user interface is used to create Skill Types.
Here, you can see listing of available Skill Types on left panel. Details of selected Skill are displayed on right panel.

Action

1. This user interface can be accessed at Configuration > Skills.
2. Edit Skill Type- Select a Skill Type on left grid, it allows User to edit it on the right panel.
3. Save Skill Type -  Click on Save Skill Type to save the change.
4. New Skill Type -  Click on New Skill Type to create a new Skill Type.
5. Undo current changes to Skill Type - Click Undo changes to cancel any changes made to selected Skill Type.
