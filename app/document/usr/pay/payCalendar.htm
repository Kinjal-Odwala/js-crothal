Payroll Calendar

The purpose of the Payroll Calendar is to maintain the Pay Periods for different Pay Frequency.

Dependencies

1. Frequency Types must be defined in a system look up table (PayPayFrequencyTypes)
2. Fiscal Patterns must be defined in a system look up table (FscPatterns)
3. Fiscal Years must be defined in a system look up table (FscYears)
