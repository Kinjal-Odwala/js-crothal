Invoice Search

The purpose of Invoice Search is to list the Invoices based on the search criteria.

UI Elements

UI Element Field Type Mandatory Source Validation
Invoice Search
Invoice # Text field No End user input Enter Invoice number.
Start Date Date Dropdown selection No End user input Start Date should be valid if entered.
Customer Text field No End user input Enter name of the Customer.
House Code Hierarchy Tree View No End user input To search a specific House Code, type-in House Code # (1455, 900000) and press Enter key.
Search Button NA End user input List the Invoicess based on the search criteria. At least one search criteria should be entered to list the Invoices.
Invoices
Invoices Grid NA NA The grid will holds the list of Invoices based on the search criteria. Click on the row in the grid to view the Invoice details.
Invoice Number Display field NA NA Number of the Invoice.
Amount Display field NA NA Total amount of the Invoice.
Customer Display field NA NA Name of the Customer.
House Code Display field NA NA House Code of the Invoice.
Invoice Date Display field NA NA Date of the Invoice.
Collected Display field NA NA Collected amount of the Invoice.
Creditted Display field NA NA Creditted amount of the Invoice.
Last Printed Display field NA NA Last Printed date of the Invoice.
