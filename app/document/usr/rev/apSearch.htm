AP Search

The purpose of AP Search is to list the AP Invoice and Check details based on the search criteria.

UI Elements

UI Element Field Type Mandatory Source Validation
AP Search
House Code Dropdown selection No End user input House Code should be selected from dropdown list.
Vendor Text field No End user input Enter Vendor Number or Name.
Invoice # Text field No End user input Enter Invoice Number.
GL Start Date Date Dropdown selection No End user input GL Start Date should be valid and should not be blank if GL End Date is entered.
GL End Date Date Dropdown selection No End user input GL End Date should be valid and should not be blank if GL Start Date is entered.
Check Date Date Dropdown selection No End user input
Check Number Text field No End user input Enter Invoice Number.
PO Number Text field No End user input Enter Invoice Number.
Search Button NA End user input List the AP Invoice details based on the search criteria. At least one search criteria should be entered to list the AP Invoices.
Export To Excel Button NA End user input Export the AP Invoice details into an excel file based on the search criteria. At least one search criteria should be entered to generate the excel file.
AP Invoices
AP Invoices Grid NA NA The grid will holds the list of AP Invoice details based on the search criteria. The list will be grouped by Vendor Invoice #. Double click on the line item to view the AP Check details. Maximum 50 records will be listed in a page and pagination will allow the user to navigate to different pages.
