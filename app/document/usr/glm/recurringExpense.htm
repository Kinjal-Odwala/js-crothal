Recurring Expenses

The purpose of Recurring Expenses is to list the Recurring Expenses based on the given criteria.

UI Layouts

Recurring Expenses

Action

1. This user interface can be accessed at Ledger > Recurring Exps.
2. Add New - Click on Add New to create new row.
3. Edit Recurring Exps - Enter House Code in House Code list and click Load button to see Recurring Expenses Info. It allows User to edit.
4. Save Recurring Exps -  Click on Save to save the change.
5. Undo - Click Undo to cancel any changes made.
