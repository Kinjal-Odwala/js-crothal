Recurring Expenses

Action

1. This user interface can be accessed at Ledger > Journal Entry.
2. Add New - Click on Add New to create new Journal Entry.
3. Edit Journal Entry - Enter House Code in House Code list and click Load button to see Journal Entries. It allows User to edit.
4. Save Journal Entry -  Click on Save to save the change.
5. Undo - Click Undo to cancel any changes made.
6. Delete - Deletes selected Journal Entry.
