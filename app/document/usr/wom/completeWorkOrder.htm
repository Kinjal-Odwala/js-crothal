Complete Work Orders

The purpose of Complete Work Orders is to update the status of the approved Work Orders as "Completed".

Dependencies

1. Transaction Status Types must be defined in a system look up table (AppTransactionStatusTypes)
2. House Codes must be defined in a system look up table (HcmHouseCodes)
3. Jobs must be defined in a system look up table (HcmJobs)
4. Jobs must be associated to the House Codes (HcmHouseCodeJobs)
5. Work Orders should be available in a look up table (WomWorkOrders)
