Epay Sites

The purpose of Epay Sites is to provide a means to setup and maintain a list of Epay Sites available. Each Epay Site can be associated to multiple House Codes.

Dependencies

1. Epay Sites must be defined in a system look up table (HcmJobs)
2. Epay Site Associations must be defined in a system look up table (HcmHouseCodeJobs)
3. House Codes must be defined in a system look up table (HcmHouseCodes)
