Action

1. This user interface can be accessed at HouseCode > Sites.
2. Edit Location - Enter Site in Sites list and click Enter button to see Billing Address and Demographics of the Site. It allows User to edit Site.
3. Save Site -  Click on Save to save the change.
3. New Site - Click New. Enter all required field details, Site Name, Address1, Postal Code, City, State, Industry Type, and Primary Business.
5. Undo current changes to site- Click Undo to cancel any changes made to selected Site.
