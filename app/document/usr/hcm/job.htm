Jobs

The purpose of Jobs is to setup and maintain the list of Jobs which are associated to the selected House Code.

UI Elements

UI Element Field Type Mandatory Source Validation / Description
Jobs Search
House Code Hierarchy Tree View Yes End user input To search a specific House Code, type-in House Code # (1455, 900000) and press Enter key.
House Code Jobs
House Code Jobs Grid NA NA List of Jobs associated to the selected House Code.
Number Display field NA NA Unique number of the Job.
Description Display field NA NA Description of the Job.
Active Display field NA NA Indicates whether the Job is active or not.
Job Details
Job Number Text field Yes End user input Unique Number of the Job.
Description Text field Yes End user input Description of the Job.
Contact Text field No End user input Contact of the Job.
Address 1 Text field No End user input Address 1 of the Job.
Address 2 Text field No End user input Address 2 of the Job.
Postal Code Text field Yes End user input Postal Code of the Job. Postal Code should be valid. Example: 99999 or 99999-9999
GEO Code Dropdown selection No End user input GEO Code of the Job. GEO codes will be listed based on the Postal Code.
City Dropdown selection Yes End user input City of the Job. City names will be listed based on the Postal Code. If City dropdown list is empty then it will allow the user to enter the city name.
State Dropdown selection Yes End user input State of the Job. State should be selected from dropdown list.
Job Type Dropdown selection No End user input Job Type of the Job. Job Type should be selected from dropdown list.
Tax Id Text field No End user input Tax Id of the Job. This field will be visible only for the "Customer" Job Type jobs. If entered, it will allow only 9 digit number.
Override Site Tax Checkbox field No End user input Override Site Tax of the Job (Y / N). This field will be visible only for the "Service Line" and "Service Location" Job Type jobs.
Service Contract Text field No End user input Service Contract of the Job.
General Location Code Text field No End user input General Location Code of the Job.
Active Checkbox field No End user input Indicates whether the Job is active or not (Y / N).
Save Button NA End user input To save the Job details and associate the Job to the selected House Code.
Undo Button NA End user input Undo the changes to Job being edited.
New Job Button NA End user input To create the new Job.
Clone Job Button NA End user input To create the new Job from existing Job templates.
Remove Button NA End user input To remove the House Code association.
Actions Menu
Save Menu item field NA End user input To save the Job details and associate the Job to the selected House Code.
New Menu item field NA End user input To create the new Job.
Undo Menu item field NA End user input Undo the changes to Job being edited.
