Action

1. This user interface can be accessed at HouseCode > Housecode Wizard.
2. Edit House Code - Enter House Code in House Code list and click Enter button to see House Code Info, Statistics, Financial/Contract, Payroll, Payroll of the House Code. It allows User to edit Site.
3. Save House Code -  Click on Save to save the change.
4. Undo current changes to House Code - Click Undo to cancel any changes made to selected House Code.
