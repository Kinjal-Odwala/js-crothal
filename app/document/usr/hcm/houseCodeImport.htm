Dependencies

1. States must be defined in a system look up table (AppStateTypes).
2. Industry Types must be defined in a system look up table (AppIndustryTypes).
3. Primary Business Types must be defined in a system look up table (AppPrimaryBusinessTypes).
4. Location Types must be defined in a system look up table (AppLocationTypes).
5. Trauma Level Types must be defined in a system look up table (AppTraumaLevelTypes).
6. Profit Designation Types must be defined in a system look up table (AppProfitDesignationTypes).
7. GPO Types must be defined in a system look up table (AppGPOTypes).
8. Ownership Types must be defined in a system look up table (AppOwnershipTypes).
9. JDE Companies must be defined in a system look up table (FscJDECompanies).
10. Service Types must be defined in a system look up table (HcmServiceTypes).
11. Service Lines must be defined in a system look up table (HcmServiceLines).
12. Remit To Locations must be defined in a system look up table (HcmRemitToLocations).
13. Contract Types must be defined in a system look up table (HcmContractTypes).
14. Terms Of Contract Types must be defined in a system look up table (HcmTermsOfContractTypes).
15. Billing Cycle Frequency Types must be defined in a system look up table (HcmBillingCycleFrequencyTypes).
16. Invoice Logo Types must be defined in a system look up table (HcmInvoiceLogoTypes).
17. Payroll Processing Location Types must be defined in a system look up table (HcmPayrollProcessingLocationTypes).
18. House Code Types must be defined in a system look up table (HcmHouseCodeTypes).
19. EPay Pay Group Types must be defined in a system look up table (HcmEPayGroupTypes).
