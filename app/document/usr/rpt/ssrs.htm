SSRS Reports

The purpose of the SSRS Reports is to load the reports and setup and maintain a list of SSRS report subscriptions.

Dependencies

1. Reports must be defined in a system look up table (RptReports)
2. Report Parameters must be defined in a system look up table (RptReportParameters)
