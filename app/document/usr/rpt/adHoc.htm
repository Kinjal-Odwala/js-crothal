Ad Hoc Report

This user interface is used to Search & Edit different Ad Hoc Reports with available application functionalities.
Here, you can see listing of available Ad Hoc Reports on First UI Layout. Details of selected Ad Hoc Report are displayed on Second UI Layout.

UI Elements

UI Element Field Type Mandatory Source Validation Description
Ad Hoc Report Search
Report List Yes End user input Please select correct report. Ad Hoc Report Name
House Code Text field Yes End user input This field is required for Search. House Code Name.
Organization Hierarchy Tree view Yes End user input Please select correct House Code. Tree view allows multi selection of nodes to be assigned to the selected Ad Hoc Report.
Search Button NA End user input Displays given House Code in Organization Hierarchy.

Filter Grid NA NA

Ad Hoc Report Grid NA End user input Please enter valid email.(for emails)

Site List Yes End user input Please select Site. Displays List of sites whose names are like the given value when ever user clicks on Enter button.
