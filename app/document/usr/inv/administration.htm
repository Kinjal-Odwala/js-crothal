Administration

The purpose of Inventory Administration is to maintain the physical inventory for each House Code. The inventory list will be shown based on the search criteria. Inventory list can be generated from Purchase Orders or Maid Cart. Count Complete flag can be updated for each Inventory.

Dependencies

1. House Codes must be defined in a system look up table (HcmHouseCodes)
2. Fiscal Years must be defined in a system look up table (FscYears)
3. Fiscal Periods must be defined in a system look up table (FscPeriods)
4. Chart of accounts must be defined in a system look up table (FscAccounts)
