Notification

The purpose of Notification is to provide a means for user to view and complete data collector tasks.

Error/Event Message

Dependencies

1. Data collector is set up from Setup/Data Collector
2. Data collector tasks are populated by schedule SQL Server agent on Monday morning (weekly data collector) or first Monday morning of the month (monthly data collector).
3. Data entry histories are stored in AppDataCollectorHistories
