Error Messages and Notifications

Error/Event Message
Save If the manadatory details are not provided, the fields will be highlighted in red.
Selection User will receive error "You must finish the oldest task of this data collector first." message if he/she selects a task of later date.

Dependencies

1. Data collector is set up from Setup/Data Collector
2. Data collector tasks are populated by schedule SQL Server agent on Monday morning (weekly data collector) or first Monday morning of the month (monthly data collector).
3. Data entry histories are stored in AppDataCollectorHistories
