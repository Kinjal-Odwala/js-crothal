Dependencies

1. When Message checkbox is checked off, Message textbox is enabled. Messagen can then be customized.
2. When Lockout checkbox is checked off, Lockout Message checkbox is checked and Lockout Message textbox is enabled. Lockout Messagen can then be customized.
3. Data collector tasks are populated by schedule SQL Server agent on Monday morning (weekly data collector) or first Monday morning of the month (monthly data collector).
4. Available modules for data collector setup are stored in AppModules
5. Available module columns for data collector setup are store din AppModuleColumns. Set AppModcActive to 0 to make the column hidden
