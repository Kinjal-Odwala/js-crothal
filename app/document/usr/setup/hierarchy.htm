Hierarchy

This user interface is used to setup Organizational Hierarchy.
Here, you can see listing of available Organizational Hierarchy nodes (treeview) on left panel. Details of selected node are displayed on right panel.
User can also search the Organizational Hierarchy for a given Unit.

Targetted User - Administrator

UI Elements

UI Element Field Type Mandatory Source Validation Description
Arbitrary Group
Active Checkbox Yes End user input This field is required. Please make an entry. Hierarchy Node is Active or Not.
Brief Text field Yes End user input This field is required. Maximum length allowed - 16 characters. If the node is of type UNIT, then it is restricted to 4 digits only. Hierarchy Node  Brief (Unit#)
Title Text field Yes End user input This field is required. Maximum length allowed - 64 characters. Hierarchy Node Title
Level List Yes End user input This field is required. Please make an entry. Hierarchy Node Level
Parent Node Treeview Yes End user input This field is required. Treeview allows selection of parent node.

Action

1. This user interface can be accessed at Setup > Hierarchy.
2. Edit Node - Select a node on the treeview on the left panel. If the node to edit is UNIT, User can search it, which expands the hierarchy to specific UNIT level. It makes easier for end User. Once node is selected, one can edit the content on the right panel.
3. New Node - Click New button. Enter all required field details, Active, Brief, Title, Level, Parent node from Treeview.
4. Change of parent node moves the node being edited under selected parent.
5. Click Save button to save the change. Undo to cancel any changes made.
6. Delete Node - Click Delete button to delete selected node.
