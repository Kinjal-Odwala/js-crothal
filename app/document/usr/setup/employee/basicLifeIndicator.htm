Basic Life Indicator

The Basic Life Indicator wizard will allow the user to modify the Basic Life Indicator of the employee and changes will not set any of the export flags. This wizard will be available only for the users or groups from having security access to the wizard.

UI Elements

UI Element Field Type Mandatory Source Validation / Description
Basic Life Indicator - General
SSN Text field Yes End user input Enter the valid Social Security Number. Example: 999-99-9999
Next Button NA End user input If SSN is valid and employee exists then it will show the Basic Life Indicator screen.
Cancel Button NA End user input Closes the Basic Life Indicator popup screen.
Basic Life Indicator - General
Basic Life Indicator Dropdown selection No End user input Basic Life Indicator should be selected from dropdown list.
Back Button NA End user input To show the Employe Search screen.
Save Button NA End user input To save the employee details.
Cancel Button NA End user input Closes the Basic Life Indicator popup screen.
