Employee Search

The Unit Manger will log into TeamFin and begin an employee search to either modify and existing employee record or look up the details on an employee.

UI Elements

UI Element Field Type Mandatory Source Validation Description
Employee Search
Last, First, SSN, or Emp# Text field NA End user input The Unit Manager will enter in either the Last Name or First Name of the Employee or the Employee SSN or the Employee #
Status Dropdown selection NA End user input The Unit Manger can also filter the search by selecting the Emp Status.
House Code Dropdown selection NA End user input House Code should be selected from dropdown Hierarchy or search HouseCode with HouseCode Brief
Search Button NA End user input List the Employees based on the search criteria
New Button NA End user input Create new user
