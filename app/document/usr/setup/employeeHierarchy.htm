Employee Hierarchy

The purpose of the Employee Hierarchy is to update the Employee Hierarchy.

UI Elements

UI Element Field Type Mandatory Source Validation / Description
Job Title Dropdown selection No End user input Job Title of Employee.
Employee Name or Number Text field No End user input To search a specific Employee, type-in Employee name and press Enter key/click Search button.
Search in Hierarchy Checkbox field No End user input Indicates whether to search in Hierarchy or not (Y / N).
Search Button No End user input Load the Node/Employee List based on the search criteria.
Employee Hierarchy - Treeview
Employee Hierarchy Nodes Treeview NA NA Lists the Employee hierarchy nodes in a treeview control.
Save Button NA End user input To save the modified node changes in original Employee Hierarchy table.
Undo Button NA End user input Undo the current changes.
Movable Employee List
Movable Employee List Treeview NA End user input Lists the searched Employees in a treeview control. Select the Employees to move to appropriate level.
Select All Checkbox field No End user input If checked, all Employees will be selected in the Movable Employee List. Only the selected Employees can be draggable and droppable at selected level.
Delete Button NA End user input Removes the selected Employees permanently.
Are you sure you want to delete the selected employee(s) permanently?
Clear Button NA End user input Removes the selected Employees from Movable Employee List.
Hierarchy - Drag & Drop
Tree Node NA NA Mouse event The selected movable Employees can be draggable and droppable at selected level. The selected Employee in a tree can be draggable and droppable at another level. The selected Employee in a tree can be draggable and droppable to movable Employee list. Nodes can be moved to its parent level node in the hierarchy tree.
