Ad Hoc Report Setup

This user interface is used to create different Ad Hoc Report with available application Modules.
Here, you can see listing of available Ad Hoc Reports on left panel. Details of selected Ad Hoc Report are displayed on right panel.

UI Elements

UI Element Field Type Mandatory Source Validation Description
Ad Hoc Reports List
REPORT TITLE Grid No NA Displays Ad Hoc Reports List.
Ad Hoc Report Details
Title Text field Yes End user input Please enter valid title. Ad Hoc Report Title.
Module List Yes End user input Please select the correct Module. Ad Hoc Report Module.
Sub Module Checkbox list selection No End user input Ad Hoc Report associated module.
Active Checkbox No End user input Ad Hoc Report is Active or Not.
Ad Hoc Report Columns
Column Order Buttons No End user input These Buttons allows user to sort selected column.
Ad-Hoc report is shown according to this Sorted List.
Ad Hoc Report Columns Grid No NA This Grid Lists all columns of Ad Hoc Report and allows
Columns to be assigned to the selected Ad Hoc Report.
EDITABLE Checkbox No End user input This Checkbox allows Ad Hoc Report column to edit.
HIDDEN Checkbox No End user input This Checkbox sets Ad Hoc Report column as Hidden.
READ ONLY Checkbox No End user input This Checkbox sets Ad Hoc Report column as Read Only.
FILTER Checkbox No End user input This Checkbox filters Ad Hoc Report according to given criteria.
Sort By Asc Checkbox No End user input This Checkbox Sorts Ad Hoc Report Column in Ascending order.
Sort By Desc Checkbox No End user input This Checkbox Sorts Ad Hoc Report Column in Descending order.
New Button NA End user input Click New button to create New Ad Hoc Report.
Save Button NA End user input Click Save button to save the change.
Undo Button NA End user input Click Undo button to cancel any changes made.
Clear Button NA End user input Click Clear button to clear all columns of Ad Hoc Report Columns Grid.

Action

1. This user interface can be accessed at Setup > Ad-Hoc Report.
2. Edit Setup - Select an Ad Hoc Report on left grid, it allows User to edit it on the right panel. Module columns are listed in right grid with multiple selection (checkbox) option.
User can select only one checkbox among EDITABLE, HIDDEN, READ ONLY checkboxes for a column. Any OPERATOR checkbox selection makes its FILTER checkbox selected.
