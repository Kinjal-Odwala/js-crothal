System Variables

This user interface is used to setup different system variables which are used through out the application. These variables carry static values and may change programatically in some cases. Example- EmployeeNumberCrothallNew gets incremented by one each time employee is created.

Here, you can see listing of available variables on left panel. Details of selected system variable are displayed on right panel.

Targetted User - Administrator

Action

1. This user interface can be accessed at menu Setup > System Variables.
2. Select a variable on left grid, it brings all details to view/edit on the right panel.
3. New - Click New button and enter all required field details like Title, Value.
4. Save - Click Save button to save the change.
5. Undo - Click Undo to cancel any changes made to selected record.
