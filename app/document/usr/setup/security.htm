Security

This user interface is used to setup different Arbitrary Groups with available application functionalities.
Here, you can see listing of available Arbitrary Groups on left panel. Details of selected group are displayed on right panel.

Targetted User - Administrator

Action

1. This user interface can be accessed at Setup > Security.
2. Edit Group - Select a group on left grid, it allows User to edit it on the right panel. Security nodes are listed in hierarchy with multiple selection (checkbox) option. Any checkbox selection makes its children disabled.
3. New Group - Click New button. Enter all required field details, Title, Group Type, Active, and security nodes from Treeview.
4. Click Save button to save the change. Undo to cancel any changes made.
5. Delete Group - Click Delete button to delete selected group.

6. In Treeview, icons with red border against each node represent that the node has children selected underneath.
