Dependencies

1. Units must be defined in a system look up table (AppUnits).
2. States must be defined in a system look up table (AppStateTypes).
3. Status Types must be defined in a system look up table (EmpStatusTypes).
4. Payroll Companies must be defined in a system look up table (PayPayrollCompanies).
5. Device Group Types must be defined in a system look up table (EmpDeviceGroupTypes).
6. Job Code Types must be defined in a system look up table (EmpJobCodeTypes).
7. Change Reason Types must be defined in a system look up table (EmpRateChangeReasonTypes).
8. Termination Reason Types must be defined in a system look up table (EmpTerminationReasonTypes).
9. Work Shifts must be defined in a system look up table (EmpWorkShifts).
10. Union Types must be defined in a system look up table (EmpUnionTypes).
11. Status Category Types must be defined in a system look up table (EmpStatusCategoryTypes).
12. Gender Types must be defined in a system look up table (EmpGenderTypes).
13. Ethnicity Types must be defined in a system look up table (EmpEthnicityTypes).
14. Federal Adjustment Types must be defined in a system look up table (EmpFederalAdjustmentTypes).
15. Marital Status Federal Tax Types must be defined in a system look up table (EmpMaritalStatusFederalTaxTypes).
16. Marital Status State Tax Types must be defined in a system look up table (EmpMaritalStatusStateTaxTypes).
17. State Adjustment Types must be defined in a system look up table (EmpStateAdjustmentTypes).
18. SDI Adjustment Types must be defined in a system look up table (EmpSDIAdjustmentTypes).
19. Local Tax Adjustment Types must be defined in a system look up table (EmpLocalTaxAdjustmentTypes).
20. Local Tax Codes must be defined in a system look up table (EmpLocalTaxCodePayrollCompanyStates).
21. Jobs must be defined in a system look up table (HcmJobs).
22. I9 Types must be defined in a system look up table (EmpI9Types).
23. Vet Types must be defined in a system look up table (EmpVetTypes).
24. Separation Codes must be defined in a system look up table (EmpSeparationCodes).
25. Job Start Reason Types must be defined in a system look up table (EmpJobStartReasonTypes).
26. Marital Status Types must be defined in a system look up table (EmpMaritalStatusTypes).
