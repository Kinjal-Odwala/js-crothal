Journal Entry Export

The purpose of Journal Entry Export to export journal information to JD. Edward.

Class

Class Name
TeamFinService

UI Elements

UI Element Field Type Mandatory Source Validation
There is no UI element

Export Process

1. Check if it is time to export
2. Retrieve Journal Entry records and update their AppTransactionStatusType by executing stored procedure GlmJournalEntryExportToJDE
3. Write Journal Entry records line by line by export file until all records are written

Configuration Settings

Configuration Key Description
ConnectionString Connection string used to connect to TeamFin database.
ExportFilePath Complete path of file folder for Journal Entry export. This folder is shared with other export processes.
ExportBackFilePath Complete path of back up file folder for Journal Entry export. This folder is shared with other export processes.
ExportAt Time when Journal Entry Export is executed

Error Messages and Notifications

Error/Event Message
Exceptions Check table AppIEExceptions for exception details (AppIECategory = 7)
Imported Number of Records Check table AppIETransactions for total exported records and execution starting/ending time (AppIECategory = 7).

Dependencies

1. Windows Services must be restarted for new configuration settings to take effect.
2. Exception and Event logging depends on LogWriter class.
