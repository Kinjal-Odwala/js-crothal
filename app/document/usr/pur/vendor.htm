Vendors

The purpose of Vendors is to provide a means to setup and maintain a list of Vendors available to the Purchasing module.

UI Elements

UI Element Field Type Mandatory Source Validation
Vendor Search
Vendor Number or Title Text field Yes End user input Enter minimun 3 characters to search the Vendors
Status Dropdown selection Yes End user input Status should be selected from dropdown list

Vendors Grid NA NA List the Vendor details based on the selected Vendor
