Items

The purpose of Items is to provide a means to setup and maintain a list of Items available to the Purchasing module.

UI Elements

UI Element Field Type Mandatory Source Validation
Item Search
Master Id, Number, Description Text field Yes End user input Enter minimun 3 characters to search the Items
Status Dropdown selection Yes End user input Status should be selected from dropdown list
Search Button NA End user input List the Items based on the search criteria
Items Grid NA NA List the Item details based on the selected Item
